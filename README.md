Дошка оголошень — веб-застосунок на Node.js з Express, EJS та Prisma, база даних SQLite.

Для запуску потрібно встановити залежності через npm install, виконати міграцію npx prisma migrate dev --name init і запустити сервер node app.js.

Змінна оточення DATABASE_URL вказує шлях до бази даних SQLite.

Структура проєкту включає папку prisma зі схемою бази даних, public зі стилями, views з EJS шаблонами та app.js з серверною логікою.

Реалізовано перегляд списку оголошень, пошук за назвою, сортування за датою та пагінацію. Оголошення відображаються у вигляді карток.

Сторінка оголошення показує повну інформацію та дозволяє видалення з підтвердженням через confirm.

Сторінка створення містить форму з валідацією даних і збереженням у базу через Prisma.
